
Ratings
========

Besucher können einzelne Seiten bewerten.


Paketname:

    quiqqer/rating


Features (Funktionen)
--------

- Star / Rating Control
- Jede Seite kann eigene Bewertungen erhalten
- Star / Rating Control besitzt Microdaten Angaben (SEO)

------------
